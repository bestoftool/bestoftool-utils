# BestOfTool Utils

A collection of privacy-first JSON utility functions. All processing is 100% client-side — your data never leaves your browser.

## Live Interactive Tools

Check out all our free live tools at: [https://bestoftool.com](https://bestoftool.com)

### Formatting
- [JSON Formatter](https://bestoftool.com/json-tools/json-formatter) — Beautify and indent JSON with syntax highlighting
- [JSON Minifier](https://bestoftool.com/json-tools/json-minifier) — Compress JSON to its smallest valid form
- [JSON Pretty Print](https://bestoftool.com/json-tools/json-pretty-print) — Format JSON with alphabetical key sorting

### Validation
- [JSON Validator](https://bestoftool.com/json-tools/json-validator) — Check JSON syntax and catch errors with exact line numbers
- [JSON Schema Validator](https://bestoftool.com/json-tools/json-schema-validator) — Validate JSON against Draft-04, 06, 07 and 2020-12 schemas

### Converters
- [JSON to CSV](https://bestoftool.com/json-tools/json-to-csv) — Export JSON arrays to Excel-ready CSV files
- [CSV to JSON](https://bestoftool.com/json-tools/csv-to-json) — Parse spreadsheet exports into structured JSON arrays
- [JSON to XML](https://bestoftool.com/json-tools/json-to-xml) — Generate SOAP-compatible XML from JSON objects
- [XML to JSON](https://bestoftool.com/json-tools/xml-to-json) — Parse SOAP, RSS, and legacy XML into JSON
- [JSON to YAML](https://bestoftool.com/json-tools/json-to-yaml) — Generate Kubernetes and DevOps YAML configs from JSON
- [YAML to JSON](https://bestoftool.com/json-tools/yaml-to-json) — Parse Kubernetes manifests and OpenAPI YAML to JSON

### Analysis
- [JSON Compare](https://bestoftool.com/json-tools/json-compare) — Instantly compare two JSON objects with color-coded output
- [JSON Diff Viewer](https://bestoftool.com/json-tools/json-diff-viewer) — Dual-mode semantic and line-by-line JSON diff tool
- [JSON Tree Viewer](https://bestoftool.com/json-tools/json-viewer) — Visualize JSON in an interactive collapsible tree

### Utilities
- [JSON Escape / Unescape](https://bestoftool.com/json-tools/json-escape) — Safely escape or unescape JSON strings

## Privacy First

We believe developer tools should not touch your data. All logic in this package runs entirely in your local environment. No server uploads, no data logging, no tracking.
